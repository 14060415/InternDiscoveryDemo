This is a demo in process, addressing the essential visual gap between the present InternDiscovery and the ongoing 630 原型系统 demo.
For our visual test purpose, only the "New Chat" page and the "SCP 广场" page are designed in detail, while the rests are presented as placeholders.
This demo offers three themes, which accommodate scenarios including formal presentation（暖白）, long-term usage（冷白）, and geek style (暗色). Users are free to switch display mode using the theme icon at the bottom of the side bar.
